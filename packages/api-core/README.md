# @ghcrawl/api-core

Internal core library for `ghcrawl`.

This package contains the Node.js runtime for:

- GitHub sync
- SQLite storage
- OpenAI embeddings
- search
- clustering
- HTTP route adapters

Most users should not install this package directly.

Use the main CLI package instead:

```bash
npm install -g ghcrawl
```
